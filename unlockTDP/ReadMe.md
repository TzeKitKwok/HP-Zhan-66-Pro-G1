# One-Key-UnlockTDP

### 安装方法

- 首次安装时运行 `install.command` 安装必要文件
- 每次唤醒后运行 `unlockTDP.command` 解锁 `TDP`
- 性能释放效果取决于机器散热情况

### 预设参数

- Junction Temperature：92 ℃
- Power Limt 2：35 W
- Power Limt 1；25 W

### 开机自启动

- Junction Temperature：92 ℃
